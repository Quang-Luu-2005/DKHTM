ESP32-CAM Live View

Nhập địa chỉ IP của ESP32-CAM để xem live stream, kiểm tra trạng thái và chụp snapshot.

Trạng thái Chưa kiểm tra Đảm bảo máy tính và ESP32-CAM cùng Wi‑Fi.

Ví dụ: http://192.168.1.50

Live stream

Chưa bật stream. Bấm Start Stream để xem camera.

Snapshot

Chưa có snapshot. Bấm Take Snapshot để chụp thử.

Endpoint nhanh

  • /status — trạng thái camera và Wi‑Fi
  • /stream — live stream MJPEG
  • /capture — trả về 1 ảnh JPEG

Bạn có thể mở file này trực tiếp trong trình duyệt hoặc dùng VS Code Live Server nếu muốn.
