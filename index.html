ESP32-CAM Live View

Nhập địa chỉ IP của ESP32-CAM để xem live stream, kiểm tra trạng thái, chụp snapshot và đăng ký khuôn mặt.

Trạng thái Chưa kiểm tra Đảm bảo máy tính và ESP32-CAM cùng Wi‑Fi.

Ví dụ: http://192.168.1.50

Live stream

Fast Stream mượt nhất; Stream + Box Balanced detect mỗi vài frame, giảm JPEG quality và bỏ delay để đỡ giật.

Chưa bật stream. Bấm Fast Stream để xem mượt nhất hoặc Stream + Box Balanced để vừa xem vừa detect.

Snapshot

Snapshot có thể detect box hoặc nhận diện manual.

Chưa có snapshot. Bấm Take Snapshot + Box để chụp thử.

Face Recognition

Nhận diện chạy dạng snapshot/manual để phù hợp ESP32-CAM thường. Khi đăng ký, chỉ để 1 khuôn mặt rõ trong khung hình.

Chưa có kết quả nhận diện. Hãy chụp snapshot, recognize hoặc enroll để xem metadata từ ESP32-CAM.

Danh sách đã đăng ký

  • Chưa tải danh sách.

Endpoint nhanh

  • /status — trạng thái camera, Wi‑Fi, PSRAM và face engine
  • /stream — fast stream MJPEG, không detect để FPS mượt nhất
  • /stream?detect=1&detectEvery=5 — stream có box cân bằng tốc độ
  • /stream?detect=1&detectEvery=1 — stream detect từng frame, chính xác hơn nhưng chậm hơn
  • /capture?detect=1 — snapshot có box khuôn mặt
  • /capture?detect=1&recognize=1 — snapshot nhận diện manual
  • /face/enroll?name=... — đăng ký khuôn mặt từ frame hiện tại
  • /face/ids — danh sách danh tính đã lưu trong flash

Bạn có thể mở file này trực tiếp trong trình duyệt hoặc dùng VS Code Live Server nếu muốn.
